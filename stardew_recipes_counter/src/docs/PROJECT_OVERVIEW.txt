🛠️ 维护指南
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   游戏更新后如何更新数据？

   1. 在游戏中按 ~ 键，执行:
      patch export Data/Objects
      patch export Data/CookingRecipes
      patch export Strings/Objects

   2. 扫描新贴图:
      python tools/list_textures.py > export_commands.txt

   3. 在 SMAPI 控制台执行导出命令

   4. 重新构建:
      ./tools/build_all.sh
