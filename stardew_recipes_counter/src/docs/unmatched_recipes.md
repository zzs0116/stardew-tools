## 如何手动补充来源

编辑 `final_recipes.json`，找到对应食谱并修改 `source` 字段：

```json
{
  "name": "杏子汁",
  "source": {
    "type": "other",
    "description": "商店购买",
    "details": {}
  }
}
```

### 可用的来源类型

| type | 图标 | 说明 |
|------|-----|------|
| `default` | 🎁 | 初始拥有 |
| `friendship` | ❤️ | NPC好感度 |
| `skill` | ⭐ | 技能等级 |
| `level` | 📈 | 玩家等级 |
| `tv` | 📺 | 电视节目 |
| `event` | 🎭 | 事件触发 |
| `seasonal` | 🗓️ | 季节限定 |
| `upgrade` | 🏠 | 农舍升级 |
| `other` | 📦 | 其他途径 |
